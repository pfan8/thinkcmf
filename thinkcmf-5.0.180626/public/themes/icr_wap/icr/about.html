什么是 I Can Read?

故事的开端，要从 I Can Read 创始人 Earnshaw 说起。Earnshaw 的孩子被诊断患有阅读障碍，学校应试的教育环境让 Earnshaw 的家庭饱受煎熬。


Earnshaw 是一位澳洲教育心理学家，作为父亲的 Earnshaw ，开始发起一项旨在探寻阅读和识字奥秘的学术研究，Earnshaw 与另一位教育心理学专家 Seargeant在这项研究中一起合作，这让 Earnshaw 看到了改变的希望。他们研究的初心，简单而美好，即能让 Earnshaw 的孩子能愉快地进行阅读。可以说，Earnshaw的孩子是 I Can Read 的第一位用户。


1995年，两人发现，阅读的痛苦其实普遍存在于大多数的学生身上。而如果有阅读障碍的孩子也可以轻松地领略阅读的奥妙，那么更多的孩子也必定能从中受益。终于，在十五年的研究之后，他们两人决定开始全新的探索——成立 I Can Read。于是，这三个简单又生动的单词，成为了两人美好愿景下的品牌烙印。


作为一项系统、科学的阅读课程，I Can Read 很快被新加坡教育部推荐，独有的特色课程走进了20个小学的课堂。如今，I Can Read的理念被越来越多的人所接受，适用于亚洲人的课程系统，让一个又一个 I Can Read 教学中心从地图上被点亮。


从 I Can Read 创立开始，十八年过去了，已经有超过25万孩子学习了 I Can Read 的课程，这是来自市场的肯定，也是 I Can Read 继续前行的动力。一个个孩子不断发出“I Can Read”的音节，那是我们彼此都倍感欣慰的时刻，也是 I Can Read 一个个新故事的延续。
